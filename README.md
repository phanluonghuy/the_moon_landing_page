# Portfolio Website - Nguyễn Thị Như Nguyệt

This is a modern, responsive portfolio website built for **Nguyễn Thị Như Nguyệt**, an Event Management professional.

## Features
- **Single Page Application (SPA):** Smooth scrolling and interactive navigation.
- **Tailwind CSS:** Modern aesthetic using the Olive, Forest, and Cornsilk color palette.
- **Mobile Responsive:** Optimized for viewing on desktops, tablets, and phones.
- **GitHub Pages Ready:** Built specifically to be hosted directly from the `/docs` folder.

## How to Host on GitHub (Free)

1.  **Upload to GitHub:** Push this entire project to a new repository on your GitHub account.
2.  **Enable GitHub Pages:**
    -   Go to your repository **Settings** on GitHub.
    -   Click on **Pages** in the left sidebar.
    -   Under the **Build and deployment** section, ensure **Source** is set to "Deploy from a branch".
    -   Under **Branch**, select `main` (or your primary branch) and change the folder from `/ (root)` to **`/docs`**.
    -   Click **Save**.
3.  **View Your Site:** GitHub will provide a URL (usually `https://username.github.io/repo-name/`) where your portfolio is live!

## File Structure
- `docs/index.html`: Main website content.
- `docs/js/main.js`: Interactive elements and scroll logic.
- `docs/assets/Nguyet_Portfolio.pdf`: The original portfolio for download.
- `docs/assets/images/`: Folder to place your profile picture and project images.

## Customization
- **Profile Picture:** Replace the placeholder in `docs/index.html` (Hero section) with an `<img>` tag pointing to your photo in `docs/assets/images/`.
- **Project Images:** Add your project photos to `docs/assets/images/` and update the `src` attributes in the Projects section.
- **Social Links:** Update the LinkedIn, Email, and Facebook links in the Footer section of `docs/index.html`.
